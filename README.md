# Wanderlust – Property Listing Platform

Wanderlust is a full stack property listing platform where users can create accounts, upload property listings with images, explore available listings and manage their properties. The platform focuses on secure authentication, image management and scalable backend architecture.

## Live Demo

(https://wanderlust-4mjx.onrender.com/listings)

## Highlights

- Secure authentication system
- Property listing management
- Image upload with Cloudinary
- Password encryption
- Responsive design
- REST API backend

## Features

- User signup/login

## Architecture

Frontend → Backend API → Database

React Client → Express Server → MongoDB

Cloudinary handles image storage.
